# 3D-Histopathology

# Build

The project can be built with [QtCreator](https://doc.qt.io/qtcreator/) or
from the command line, with [qmake](https://doc.qt.io/qt-5/qmake-manual.html)
```bash
mkdir build
cd build
qmake ../3d_raycaster.pro
make
./3d_raycaster
```
